# Kite JBang Catalog

[JBang](https://jbang.dev) catalog for [Kite CLI](https://github.com/kitecorp/kite-cli).

## Installation

```bash
# Run directly
jbang kite@kitecorp --version

# Install as an app
jbang app install kite@kitecorp

# Then run
kite --version
```

## Requirements

- Java 25 or later
- JBang

## Available Aliases

| Alias | Description |
|-------|-------------|
| `kite` | Multi-cloud Infrastructure as Code CLI |
